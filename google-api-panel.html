📋 API Key Alma Rehberi

Google Cloud Console'dan API Key Alma

  1. console.cloud.google.com adresine gidin
  2. trgokmenyilmaz@hotmail.com ile giriş yapın
  3. "Yeni Proje" → "yapigucbeton" adını verin → Oluştur
  4. Sol menü → "API ve Hizmetler" → "Kitaplık"
  5. Şu API'leri etkinleştirin: Maps JavaScript API, PageSpeed Insights API, Indexing API, Translate API
  6. "Kimlik Bilgileri" → "+ Kimlik Bilgileri Oluştur" → "API Anahtarı"
  7. Oluşturulan KEY'i Office Agent'a gönderin → Otomatik kurulum yapılır!
